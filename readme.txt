Aitasi Coming Soon - A fully Responsive coming soon landing page and install in your WordPress site quickly and easily.

== Description ==

Aitasi Coming Soon is a modern, beautiful, responsive, and full-width professional landing page that’ll help you create stunning Coming Soon Page or Maintenance Mode pages instantly without any coding or design skills. You can work on your site while visitors see a Coming Soon or Maintenance Mode page.


= Features = 
* 100% Fully Responsive.
* Workable with any WordPress Theme.
* Visible only to non-logged-in users or visitors.
* Highly customizable, flexible, and easy to use.
* Create a stunning coming soon landing page instantly.
* Set countdown time easily from the plugin’s setting.
* Unlimited colors and styling options.
* Social profiles & social sharing buttons.
* Add social profiles to your coming soon page.
* Full browser backgrounds option.
* And much more options.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for ‘Aitasi Coming Soon’
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to Settings > Aitasi Settings to configure the plugin

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select ‘aitasi-coming-soon.zip’ from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to Settings > Aitasi Settings to configure the plugin

= Using FTP =

1. Download ‘Aitasi Coming Soon’ Plugin
2. Extract the ‘aitasi-coming-soon’ directory to your computer
3. Upload the ‘aitasi-coming-soon’ directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
5. Go to Settings > Aitasi Settings to configure the plugin
